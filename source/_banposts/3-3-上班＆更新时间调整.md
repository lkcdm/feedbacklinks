# 前言

很久没更新了，为什么呢？

身为一个快步入社会的牲畜人，我决定提前体验一下自己打工赚钱，同时，我的众多公益项目，因为急缺资金，所以说也算是弄的一点备用金吧，我也想换台设备。

于是乎，我寻求我母亲给我找了一份工作，她是问了她的经理收人吗，然后我就进去了，面试那天十分梦幻，很多人，但是只收7人，最后一个人选经理问我们：“你们谁做过工”然后我母亲就说我以前在茶楼打工过（我大姑，她总经理我进去体验生活）于是我就被录用了，厉害

![IMG_20240712_082149](https://koimg.cn-sy1.rains3.com/2024/07/14/794450.webp)

# 工作解释

这是一个生产衣服的地方，踩缝纫机生产加工一系列的生产活动

![IMG_20240712_074710](https://koimg.cn-sy1.rains3.com/2024/07/14/020349.webp)

录用之后，我也是可以直接上手开始工作，我母亲给我报的貌似是模板工，一个月3800-4000多人民币的工作，并且包吃，很香，也算是不错了，不多，但是如果一个人生活什么的也够用，成功击败三线城市以外的大部分打工人~（开玩笑我主要是听说很多人月薪3千不包吃住）

经过了一系列的教程，组长简单的教学了我一下我的工作流程，期间闹了一点乌龙，就是组长教学错误了哈，还是经理过来说的纠正的，怪尬尴的

![IMG_20240712_113607](https://koimg.cn-sy1.rains3.com/2024/07/14/381623.webp)

我的工作就是烫压衣服，不是说平时你们大理用的烫压板，就是压衣服两肩和衣领的连接处，使其整齐并美观，同时我们这个也是比较末尾的工作流程，搞砸一件衣服估计要掉不少肉，我工作的这三天加工的是一批很赶的货，一键休闲服，貌似是老人穿的或者是出口的，一件衣服要1399人民币，神仙了

![IMG_20240712_130248](https://koimg.cn-sy1.rains3.com/2024/07/14/863272.webp)

打累了就停下来歇会摸鱼，主要是这职位就是站着脚很累，比中学开学那几天军训累十倍，简直了，工作是十分乏味的，所以说我在工作的时候找了十分多的方法，比如说平常是两件衣服一轮，我直接4件双倍效率，或坐下来操作但是效率会慢，我一般是打两件坐下来打一件

<img src="https://koimg.cn-sy1.rains3.com/2024/07/14/688914.webp" alt="IMG_20240712_120556" style="zoom:33%;" />

因为包吃，所以说中午12下班就排队吃饭，十分辛苦，我站了几个小时了这比上课还难受，毕竟在工作中可没法打瞌睡偷懒什么的，还有达标量的！

<img src="https://koimg.cn-sy1.rains3.com/2024/07/14/349199.webp" alt="Image_1720757428500" style="zoom:33%;" />

还是很香的，比我厨艺好多了（废话），当然我是和我母亲两个人，所以说菜可能不是很多，所以我一直是一口菜三口饭，不过母亲还是剩了好多菜给我，还是我一个人吃了，母爱伟大

# 工作结束

经历了12个小时的工作晚上是终于下班的，我们家距离工作的地方还是很远的，最少都要半小时了

<img src="https://koimg.cn-sy1.rains3.com/2024/07/14/027289.webp" alt="IMG_20240712_214732" style="zoom:33%;" />

到家了，就撸猫会，睡觉了，麻烦的早八，现在知道为什么这么多打工人天天赶脚早八了

# 人生劝诫

现在是7月14日，我工作的第三天，我这一轮下来可谓是精神+肉体的折磨，十分痛苦，精神上被困死，肉体上脚感觉上是被人砍了，又麻又痛，以前花钱只觉得这点钱而已少的很，花的很自在逍遥，到现在，自己去干活了，一个小时的钱是12.8元，才知道赚一点点钱是多么的困难，在我没体验之前一直以为打工容易，我在这里十分劝诫大伙
